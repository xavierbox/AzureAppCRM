# Azure Async Simulator 

Project to build, step by step, an Azure-based system for launching long-running simulation jobs.

# Azure services used 
- Azure identities
- Managed identities
- RBAC and permissions
- Azure App Service
- Azure Functions
- Azure Storage Accounts
- Azure Blob Storage
- Azure Queues
- Container-based workers
- Async job orchestration patterns
- 
# Repository structure

- `docs/`  
  Notes, explanations, and learning checkpoints.

- `src/`  
  Application code as it starts to appear.

- `infra/`  
  Infrastructure-related material such as Azure CLI snippets, templates, and setup notes.

- `experiments/`  
  Small isolated tests used to learn a specific Azure feature.
